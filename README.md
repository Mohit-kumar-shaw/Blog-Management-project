# Blog Management System

A full-stack Blog Management System built with Node.js, Express.js, and EJS that allows users to create, edit, delete, and manage blog posts efficiently with dynamic server-side rendering.

---

## Features

- Create new blog posts
- Edit existing blog posts
- Delete blog posts
- View all blogs dynamically
- Server-side rendering using EJS
- Responsive and user-friendly interface

---

## Tech Stack

- Node.js
- Express.js
- EJS
- HTML5
- CSS3
- JavaScript

---

## Installation

1. Clone the repository

```bash
git clone https://github.com/Mohit-kumar-shaw/Blog-Management-project.git
```

2. Navigate to the project folder

```bash
cd blog-management-system
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
node index.js
```

5. Open in browser

```bash
http://localhost:3000
```
